Parkinson's Disease Prediction App

A Machine Learning based web application that predicts the likelihood of Parkinson’s Disease using biomedical voice measurement data.
The application uses trained machine learning models to analyze input parameters and provide real-time predictions through a simple and interactive web interface.

Features
Parkinson’s Disease prediction using Machine Learning
User-friendly web interface
Real-time prediction system
Data preprocessing and feature scaling
Trained and optimized ML models
Fast and accurate prediction results
Easy setup and deployment

Installation
1. Clone the Repository
git clone https://github.com/Shikhar1330/parkinsons_app.git
cd parkinsons_app
2. Create Virtual Environment
python -m venv venv
3. Activate Virtual Environment
Windows
venv\Scripts\activate
Linux / Mac
source venv/bin/activate
4. Install Dependencies
pip install -r requirements.txt
Run the Application
python app.py

The application will run locally at:

http://127.0.0.1:5000
